Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with test-task API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API use jwt. You need to get token by POST to /api/auth with login and password.
Next you should add header Authorization : Bearer token

Endpoints

Invoke jwt token

Example request:
curl --request DELETE \
    "http://localhost/api/auth" \
    --header "Authorization: Bearer token" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/auth"
);

const headers = {
    "Authorization": "Bearer token",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "DELETE",
    headers,
}).then(response => response.json());

    "message": "User logged out successfully"
}

Get a list of users

Only users with admin role can use this endpoint

Example request:
curl --request GET \
    --get "http://localhost/api/users" \
    --header "Authorization: Bearer token" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"limit\": 3,
    \"page\": 83,
    \"name\": \"debitis\",
    \"email\": \"bergnaum.russel@example.net\"
}"

const url = new URL(
    "http://localhost/api/users"
);

const headers = {
    "Authorization": "Bearer token",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "limit": 3,
    "page": 83,
    "name": "debitis",
